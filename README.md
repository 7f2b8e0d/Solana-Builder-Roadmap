# 🌐 Solana 全栈大师项目：DecentraStack

从零基础到商业级 dApp 全栈开发专家的完整路线图
（贯穿 Solana 全部核心知识点）

---

## 📘 项目介绍

DecentraStack 是一个 **从极简到复杂、持续演进的 Solana 全栈训练项目**。
目标是让你从零开始，逐步掌握：

* Solana 底层运行机制
* Rust + Anchor 智能合约
* PDA、CPI、SPL Token、NFT、Marketplace、Escrow
* 链下服务、索引器、数据库、后端
* React/Vue dApp 前端
* 部署、安全审计、优化

整个路线由 **6 大篇章、18 个阶段** 组成，每一阶段都是完整可运行的小项目，能自然演化到一个完整商业级应用。

最终成果是一款真实的 Web3 产品：

### 🌐 **DecentraStack：链上商城 + NFT 权限 + 支付托管 + 后台系统 + 前端 dApp**
